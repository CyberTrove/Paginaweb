Las herramientas de seguridad funcionan como una barrera que proteje al usuario de cosas como malware,spyware,robos de datos,scripts maliciosos,etc... O en otros casos sirven para auditorias de seguridad, que son pruebas que se realizan para saber si un sistema es seguro o no. Y no solo sirven para defenderse, sino que tambien sirven para atacar,
